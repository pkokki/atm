profile settings

When a profile setting is set at more than one profile, the value entered at the global profile is assigned the lowest priority and is superseded by any value entered at a solution profile. Any value entered at a claim profile will supersede solution profile settings. The value assigned at a user profile is given the highest priority, overriding any values entered at other profile.
